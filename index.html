COIN is designed to promote the development of a new family of tools for data exploration in cosmology. </p> </blockquote> <h2 id="toc_1">Generalized Linear Models in Astronomy</h2> <p>Statistical methods play a central role to fully exploit astronomical catalogues and an efficient data analysis requires astronomers to go beyond the traditional Gaussian-based models. This projects illustrates the power of generalized linear models (GLMs) for astronomical community, from a Bayesian perspective. Applications range from modelling star formation activity (logistic regression), globular cluster population (negative binomial regression), photometric redshifts (gamma regression), exoplanets multiplicity (Poisson regression), and so forth.</p> <h3 id="toc_2">Binomial Regression</h3> <p><a href="http://adsabs.harvard.edu/abs/2014arXiv1409.7696D" class="btn btn-primary">Link to ADS</a> <a href="#" class="btn btn-primary">Tutorial</a></p> <h3 id="toc_3">Gamma Regression</h3> <ul> <li><p><a href="http://adsabs.harvard.edu/abs/2015A%26C....10...61E">Paper</a></p></li> <li><p><a href="http://ascl.net/1408.018">Package</a></p></li> <li><p><a href="http://cosmophotoz.readthedocs.org/en/latest/">Tutorial</a></p></li> <li><p><a href="https://cosmostatisticsinitiative.shinyapps.io/CosmoPhotoz">Web-interface</a></p></li> </ul> <h2 id="toc_4">Aproximate Bayesian Computation</h2> <p>Approximate Bayesian Computation (ABC) enables the statistical analysis of stochastic models for complex physical systems in cases where the true likelihood function is unknown, unavailable, or computationally expensive. ABC relies on the forward simulation of mock data rather than the specification of a likelihood function. The CosmoABC code was originally designed for cosmological parameter inference from galaxy clusters number counts based on Sunyaev-Zel’dovich measurements. Nevertheless, the user can easily take advantage of the ABC sampler along with his/her own simulator, as well as test personalized prior distributions, summary statistics and distance functions.</p> <ul> <li><p><a href="https://pypi.python.org/pypi/CosmoABC">Package</a></p></li> <li><p><a href="http://cosmoabc.readthedocs.org/en/latest/">Tutorial</a></p></li> </ul> <h2 id="toc_5">Analysis of Muldimensional Astronomical DAtasets (AMADA)</h2> <p>AMADA allows an iterative exploration and information retrieval of high-dimensional data sets. This is done by performing a hierarchical clustering analysis for different choices of correlation matrices and by doing a principal components analysis in the original data. Additionally, AMADA provides a set of modern visualization data-mining diagnostics. The user can switch between them using the different tabs.</p> <ul> <li><p><a href="http://rafaelsdesouza.github.io/AMADA/">Package</a></p></li> <li><p><a href="https://cosmostatisticsinitiative.shinyapps.io/AMADA/">Web App</a></p></li> </ul> <hr> <h4 id="toc_6">Contact: <a href="mailto:rafael.2706@gmail.com">rafael.2706@gmail.com</a></h4> <p>
