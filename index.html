This projects illustrates the power of generalized linear models (GLMs) for astronomical community, from a Bayesian perspective. Applications range from modelling star formation activity (logistic regression), globular cluster population (negative binomial regression), photometric redshifts (gamma regression), exoplanets multiplicity (Poisson regression), and so forth.</p> <h3> <a id="binomial-regression" class="anchor" href="#binomial-regression" aria-hidden="true"><span class="octicon octicon-link"></span></a>Binomial Regression</h3> <ul> <li><p><a href="http://adsabs.harvard.edu/abs/2014arXiv1409.7696D">Paper</a></p></li> <li><p><a href="">Tutorial</a></p></li> </ul> <h3> <a id="gamma-regression" class="anchor" href="#gamma-regression" aria-hidden="true"><span class="octicon octicon-link"></span></a>Gamma Regression</h3> <ul> <li><p><a href="http://adsabs.harvard.edu/abs/2015A%26C....10...61E">Paper</a></p></li> <li><p><a href="http://ascl.net/1408.018">Package</a></p></li> <li><p><a href="http://cosmophotoz.readthedocs.org/en/latest/">Tutorial</a></p></li> <li><p><a href="https://cosmostatisticsinitiative.shinyapps.io/CosmoPhotoz">Web-interface</a></p></li> </ul> <h2> <a id="aproximate-bayesian-computation" class="anchor" href="#aproximate-bayesian-computation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Aproximate Bayesian Computation</h2> <ul> <li><p><a href="https://pypi.python.org/pypi/CosmoABC">Package</a></p></li> <li><p><a href="http://cosmoabc.readthedocs.org/en/latest/">Tutorial</a></p></li> </ul> <h2> <a id="analysis-of-muldimensional-astronomical-datasets-amada" class="anchor" href="#analysis-of-muldimensional-astronomical-datasets-amada" aria-hidden="true"><span class="octicon octicon-link"></span></a>Analysis of Muldimensional Astronomical DAtasets (AMADA)</h2> <p>AMADA allows an iterative exploration and information retrieval of high-dimensional data sets.
